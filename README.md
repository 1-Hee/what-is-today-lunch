## 🔍 프로젝트 개요 (Project Overview)
오늘 점심은 뭐지? 매번 물어보기 귀찮으셨죠? 우리 서비스는 단체, 기관에서 반자동으로 점심 메뉴를 알려드립니다.
점심 메뉴 공유 서비스의 위키백과!, 사용자 참여로 운영되는 점심 메뉴 공유 서비스, 당신의 점심 시간을 즐겁게 해드립니다.
더불어 오늘의 점심에 대한 호응도를 파악 가능한 레이팅 시스템까지! 이젠 맛있는 점심과 함께 스낵 같은 우리 서비스를 즐겨보세요 🎉
